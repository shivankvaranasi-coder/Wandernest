# Wandernest
Hello Wandernest is a travel booking website which help us to compare the plans of different destination according to your convience so that next time you travel you travel with peace.The platform provides an intuitive interface for browsing destinations, checking availability, and making bookings online.So for you Wanderlust we have Wandernest.
Features

🔍 Search travel destinations

🏨 View hotels and travel packages

📅 Check availability and booking dates

💳 Easy booking system

👤 User registration and login

📱 Responsive design for mobile and desktop

⭐ Reviews and ratings for destinations
Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: Node.js ,Express Js

Database:MongoDB

Version Control: Git & GitHub
